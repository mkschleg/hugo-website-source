The Dirac delta can be thought of as a function on the real number line which is zero everywhere except the origin:

\\[
\delta(x)=\left\\{\begin{array}{ll}
+\infty, & x=0 \\\\
0, & x \neq 0
\end{array}\right.
\\]

and which is also constrained to satisfy

\\[\int\_{-\infty}^{\infty} \delta(x) d x=1.\\]

While this is not well defined through this heuristic, this can be re-characterized as a measure.

The Dirac Measure is a measure \\(\delta\_x\\) on a set \\(X\\) (with any $&sigma;$-algebra) defined for a given \\(x\in X\\) and any measurable set \\(A \subset X\\) by

\\[
\delta\_{x}(A)=1\_{A}(x)=\left\\{\begin{array}{ll}
0, & x \notin A \\\\
1, & x \in A
\end{array}\right.
\\]
