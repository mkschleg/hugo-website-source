A **stationary point** of a function is the value in its domain such that the [derivative]({{< relref "calculus.md#single-variate-derivatives" >}}) (or [gradient]({{< relref "calculus.md#the-gradient-operator" >}})) is zero (or the zero vector). This is slightly different from a **critical point** where the set of critical points contains stationary points and also includes points where the derivative is not-defined. When the derivative is undefined, the point is also called a **singular point**.

<div title="Stationary Points" class="definition">

The subset of a function's domain \\(X\\) (for function \\(f: X \rightarrow \numfield\\)) whose derivative \\(f^\prime( c) = 0; c \in X\\) is called a **stationary point**.

</div>

<div title="Singular Points" class="definition">

The subset of a function's domain \\(X\\) (for function \\(f: X \rightarrow \numfield\\)) where the function is non-differentiable (i.e. the derivative is undefined) is called a **singular point**.

</div>

<div title="Critical Points" class="definition">

The set of points containing a union between the singular points and stationary points are known as the **critical points** of a function.

</div>
