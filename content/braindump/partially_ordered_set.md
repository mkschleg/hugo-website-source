A partially ordered set (poset) formalizes and generalizes the intuitive concept of an ordering, sequencing or arrangement of the elements of a set. A poset consists of a set with a binary relation (or comparator) over which elements are arranged. While in a normal "ordered set" all elements are comparable, it is possible for a pair of elements in the set where neither takes precedence. A partial order generalizes a total order.

See [link](https://en.m.wikipedia.org/wiki/Partially_ordered_set) for further information.
